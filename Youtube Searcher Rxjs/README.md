# Youtube Searcher app (Angular 2.0 beta.11) 
- using rxjs to enhance searching mechanism.
- using http for get request.
- using youtube api to search videos through keywords (Note: Youtube api credentials are reuired to access)
  for further information visit https://developers.google.com/youtube/v3/docs/search/list

# Instructions to run project
1)Type npm install from the root folder in cmd
2)Then type npm start
2)You are all set to go now Enjoy!
